# Mobile App

Expo React Native client for the SMS project.

## Commands

```bash
npm install
npx expo start
```

Platform shortcuts:

- `npm run ios`
- `npm run android`
- `npm run web`
- `npm run lint`

## Structure

- `App.js`: app entry and navigation
- `src/screens`: role-based screens
- `src/components`: shared UI
- `src/api.js`: API client
- `src/ui/theme.js`: shared theme tokens
